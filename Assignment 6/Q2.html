Rules for BEM

  • names are written in lowercase
  • words are seperated by a hypen(-)
  • the block name defines the namespace for its elements and modifiers
  • The element name is seperated from the block name by a double underscore
  • The modifier name is seperated from block or element name by two double underscore(__)
  • The modifier name is seperated from the block or element name by double dashes(--)
